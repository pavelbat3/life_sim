# life_sim
Проект "Консольный симулятор жизни"


Простой консольный симулятор жизни — система, в которой существуют организмы, взаимодействующие друг с другом и окружающей средой.

Возможные идеи:

● Симулятор популяции животных и растений

● Модель "хищник-жертва"

● ИИ-агенты, принимающие решения

● Пошаговая игра с эволюцией

Требования к проекту:

● Минимум 3 класса.

● Чётко разделённые модули.

● Использование Git для командной работы.

● Документация и комментарии.

● Соответствие PEP8.

● Тестирование базовой логики.
